Github Issue Kanban
==========

Kanban board of github issues.


Development Setup
--

Clone repo and run bundle install.

```
$ git clone git@github.com:amolbrid/kanban.git
$ cd kanban
$ bundle install
```

Start server:

```
$ bundle exec rails s
```
